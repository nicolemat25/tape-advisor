# Tape Advisor

Asistente personal de tape reading basado en el bootcamp completo.

## Desplegar en Vercel

1. Sube esta carpeta a GitHub (ver instrucciones abajo)
2. Conéctala en vercel.com
3. Añade la variable de entorno: VITE_ANTHROPIC_KEY = tu API key

## Obtener API Key
- Ve a console.anthropic.com
- Crea una cuenta
- API Keys → Create Key
